# Flutter Weather App

> **This project was built to test Opus 4.7's coding capabilities.** The entire app — architecture, UI, animations, and CI/CD — was generated by Claude Code (Opus 4.7) with minimal human guidance.

A Xiaomi Weather-inspired Flutter weather application featuring immersive full-screen experience, dynamic weather animations, GPS-based location detection, and rich weather data visualization.

## Features

- 🌤️ Immersive full-screen weather experience
- 🎨 Dynamic weather animations (rain, snow, thunderstorm, fog, etc.)
- 📍 GPS-based automatic location detection
- 📊 Detailed weather metrics (temperature, humidity, wind, UV, etc.)
- 🌡️ Multi-day forecast & hourly breakdown
- 🌅 Sunrise / sunset with animated arc
- 🫧 Air quality index (AQI) display
- 🔍 City search with autocomplete

## Tech Stack

- **Flutter** (cross-platform: Android & iOS)
- **Open-Meteo API** — free, no API key required
- **Geolocator** — GPS positioning
- **GitHub Actions** — automated Android APK builds

## Data Source

[Open-Meteo](https://open-meteo.com/) — Free non-commercial weather API, no registration needed.

## Building

```bash
flutter pub get
flutter run
```

Android APK builds are handled automatically via GitHub Actions — no local Android SDK required.
